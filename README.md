# Plan It

A modern travel planning dashboard for multi-city trips. Built with Next.js, React, TypeScript, Tailwind CSS, and Supabase.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Command Center**: Dashboard overview with summary metrics and quick navigation
- **Timeline View**: Visual horizontal timeline showing cities, lodgings, and transport
- **Day-by-Day Itinerary**: Detailed daily plans organized by time blocks
- **Task Management**: Action items with priorities, due dates, and entity linking
- **Parties Management**: Coordinate travel groups with different date ranges
- **Logistics Hub**: Centralized management of transport, lodging, and reservations
- **Budget Tracking**: Multi-level expense tracking with actual vs. estimated costs
